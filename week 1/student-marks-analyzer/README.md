# 📊 Student Marks Analyzer (NumPy Project)

A simple Python project using NumPy to analyze student marks.

## 🚀 Features

- Student-wise average
- Subject-wise average
- Highest & lowest marks per subject
- Top student detection
- Grace marks addition
- Pass/Fail detection
- Grade assignment system

## 📦 Technologies Used

- Python
- NumPy

## ▶️ How to Run

```bash
python main.py
