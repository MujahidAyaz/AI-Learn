# Student Performance Predictor Using Polynomial Regression

A beginner Machine Learning project built from scratch using NumPy.

## Features

- Polynomial Regression (Degree 2)
- Gradient Descent
- Mean Squared Error
- Data Visualization
- Prediction System

## Dataset

| Study Hours | Marks |
|------------|--------|
| 1 | 5 |
| 2 | 12 |
| 3 | 23 |
| 4 | 38 |
| 5 | 57 |

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Run

```bash
pip install -r requirements.txt
python polynomial_regression.py
```

## Learning Goals

- Understand Polynomial Regression
- Understand Gradient Descent
- Understand Feature Engineering
- Understand Overfitting vs Underfitting
